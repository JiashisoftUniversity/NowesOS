Jiashisoft Community 高等开源项目

《NowesOS V2.0 ClassWorld 92012》

NowesOS是一款具有自主国产UI系统的便捷全能U盘启动盘，一个小巧的口袋系统，就像一个RAMOS，本版本为64位（USB）版本。
这是原贾氏科技（现韦德软件）既自2010的NowesUI迭代三次与2014年的Nowes1.0与XF版本后，经过近8个月的准备与开发、如期在2015年8月发布的新一代操作系统版本。

・目录说明
-.Common 存放公用模块与支持库
-.Core 存放核心程序代码，如桌面，关机重启，搜索，任务管理，欢迎，任务栏
-.Lib 存放公共动态链接库文件，供其它程序运行时使用，调试时最好编译好放在里面
-.Main 存放了系统四大金刚：应用商店软件平台，控制设置面板，文件管理器与网络浏览器
  特别注意：浏览器 JunlySafe是作为独立项目开源的，可在本项目隔壁找到
-.Setup 存放系统安装器，考虑到这个的特殊性，只公开了表层代码部分，底层如写引导暂不提供
-.Tool 存放了系统软件平台的APP打包工具，可以利用此工具快速给Nowes扩展许多应用程序与游戏，只需要简单执行绿化。
  特别注意：由于AppMaker是使用了Jiashisoft Research的专利技术，所以源码暂不公开

・开源版二次开发说明：
-开源部分为系统的UI，而其中包含的系统封装技术，驱动技术，引导技术则出于协议不得被公开。
-可以使用grub的镜像作为母盘，也可以使用bootmgr/efi技术，需要手动修正指向内核文件。
-系统UserInterface，非独立发行的，不要将程序放入system文档夹，以免报错。
-非AIK母盘不要使用KILL EXPLORER命令来解决内核与系统协调问题，会导致软件Runtime取不到参数。
-图标问题与软件更新问题、以及小BUG可以参见源代码给予修正和更改。
-NAP是一个良好的口袋系统的软件扩展平台，但对应软件所需注册表必须每次重启后导入一次。

・ClassWorld开发
    6.16初稿->7.26公测->9.20正式

・系统启动过程：
   set->bios->0x0->loader->grub/bootmgr->init->kernel->ntoskrnl->mount->login->sync->welcome->desktop->startbar->msg

・配置要求：
-推荐 2GB 以上运存
-双核 CPU 1Ghz或更快
-兼容性高的显卡和声卡

・安装方法
1.安装到硬盘
把ISO解压到某个你喜欢的硬盘，[比如]E盘、然后到E盘点Setup安装就行
最好系统是WIN7以上，不然可是要写硬盘引导的哦！
安装后的系统就在E盘，用户文件在E盘的User下。
2.安装到U盘
使用 ULTRAISO 打开ISO，并在菜单中找到 启动 > 写入硬盘镜像 ，然
后选择你的U盘，点写入即可
在BIOS中更改引导顺序，将U盘放在第一就可以了
注：若要在live模式下安装到硬盘 请在电脑由U盘启动后 复制文件到硬盘 找到Setup以安装

・系统优点
1.基于WindowsNT 6.3 软件平台 64位 高兼容性
2.每次启动相当于一次恢复，正常启动情况下病毒无法渗透系统
3.使用 RAMDISK 技术虚拟系统盘，使得系统读写速度以GB计速
4.易用的NowesOS 商店生态、支持快速转化APP到NowesOS
5.内置多款实用软件 内测版默认激活 为授权版 现在免费
6.内置高效信息中心，智能提高综合素质

・系统缺点
1.暂不支持任何程序调用API操作系统程序开机启动
2.声卡与显卡等驱动不稳定/空白 且无法自动化加载
3.部分软件还需更新 应用商店平台仍需修复
4.小BUG如与explorer协调 重复运行 按钮失效等
